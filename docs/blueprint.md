# **App Name**: SalamConnect

## Core Features:

- User Authentication: Secure user registration and login with Firebase Authentication, including email/password and Google sign-in. Private routes restrict access to logged-in users only.
- Profile Creation and Management: Users can create and customize their profiles with avatars, cover photos, bios, and editable profile information.
- Social Feed: Display posts from followed users in reverse chronological order, with interactive elements for likes, comments, and shares.
- Content Moderation: Use AI as a tool to identify and filter inappropriate content. Ensures a safe environment. 
- Audio Rooms: Enable users to create and join audio rooms for live discussions, featuring speaker/listener roles and request-to-speak functionality.
- Reels Playback: Implement a full-screen vertical video feed with auto-play and snap scrolling for short video clips.
- Groups (Majlis): Allow users to create and join groups, facilitating community engagement around shared interests. Support role-based access control (owner, admin, member).

## Style Guidelines:

- Primary color: Sky Blue (#007BFF) for main actions and interactive elements, evoking trust and clarity.
- Accent color: Gold (#FFD700) for highlights and key brand elements, drawing inspiration from Abbasid aesthetics and conveying sophistication.
- Background color: Very light grayish-blue (#F0F8FF) for a clean, modern look. Provides a neutral backdrop that highlights the primary and accent colors.
- Arabic font: 'Cairo' (sans-serif) for the Arabic text.
- English font: 'Poppins' (sans-serif) for the English text. Note: currently only Google Fonts are supported.
- Implement a responsive design with a persistent left sidebar on desktop and a bottom navigation bar on mobile for primary navigation.
- Support right-to-left (RTL) layout as the default, with consideration for both light and dark themes.
